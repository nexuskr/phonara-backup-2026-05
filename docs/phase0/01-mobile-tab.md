# PHONARA V2 — Mobile Tab별 상세 화면 설계

**목표**: 한 손으로 조작 가능하고, 감정적으로 자극적이며, FOMO가 지속적으로 작동하는 모바일 중심 UX

---

## 1. 전체 Bottom Navigation 구조 (5탭)
[ 홈 ]     [  벌기  ]     [  게임  ]     [  소셜  ]     [  프로필 ]
text- **엄지 존 최적화**: 하단 고정, 큰 터치 영역
- **전환 애니메이션**: Framer Motion spring (부드럽고 빠름)
- **각 탭 진입 시**: 즉시 로딩되는 느낌 (skeleton 최소화)

---

## 2. 탭별 상세 화면 설계

### 탭 1: 홈 (Home)

**주요 목적**: 
- 오늘의 보상 상태 확인
- FOMO 자극
- 다음 행동 유도

**화면 구성**:

1. **상단 헤더**
   - 현재 PHON 잔고 (실시간 count-up)
   - 스트릭 불꽃 표시 + 연속 일수

2. **오늘의 보상 카드**
   - “오늘 출석 보상 받기” 버튼 (크고 명확)
   - 이미 받은 경우: “내일 다시 만나요 ”

3. **실시간 FOMO 피드** (마키 또는 카드 리스트)
   - “방금 전 ‘김민수’님이 3일 스트릭 달성!”
   - “지금 187명이 미션 진행 중”

4. **추천 미션 / 한정 미션**
   - “오늘만 2배 보상” 배지
   - 빠르게 완료 가능한 미션 2~3개

5. **하단 CTA**
   - “게임으로 더 벌어볼까요?” 버튼 → 게임 탭 이동 유도

**심리 설계**:
- 즉시성 (오늘 보상)
- FOMO (실시간 활동)
- 다음 행동 유도

---

### 탭 2: 벌기 (Earn)

**주요 목적**:
- 보상 수령의 주된 공간
- 성취감 극대화

**화면 구성**:

1. **상단 요약**
   - 오늘 획득한 PHON
   - 이번 주 총 획득량

2. **출석 / 스트릭 섹션**
   - 큰 캘린더 스타일 or 연속 불꽃
   - “7일 연속 출석 시 보너스” 강조

3. **미션 리스트**
   - Daily Missions (3~5개)
   - Limited Time Missions (타이머 표시)
   - 각 미션에 “보상 XX PHON” + 진행도

4. **미스터리 박스**
   - “오늘의 미스터리 박스 열기” (광고 or 간단 행동 후 오픈)
   - 기대감 조성

5. **바이럴 태스크**
   - “친구 1명 초대하기” 
   - “공유하고 추가 보상 받기”

**심리 설계**:
- 성취감 (미션 완료)
- 기대감 (미스터리 박스)
- 사회적 증거 + 바이럴

---

### 탭 3: 게임 (Game)

**주요 목적**:
- 재미 + 즉시 보상
- TikTok-style 가벼운 플레이

**화면 구성**:

1. **상단**
   - “오늘 게임으로 번 PHON” 표시

2. **게임 카드 리스트**
   - V1: Keepy-Uppy (축구공 튀기기)
   - 추후: 간단 슬롯, 크래시 등
   - 각 게임에 “평균 보상”, “최고 기록” 표시

3. **게임 진입 전**
   - 간단한 설명 + 예상 보상
   - “지금 플레이하면 평균 XX PHON”

4. **플레이 후**
   - 즉시 결과 + 보상 지급 애니메이션
   - “다시 도전하기” 버튼 강조 (중독성)

**심리 설계**:
- 즉시 보상
- 재도전 유도
- 가벼운 재미 (TikTok 감성)

---

### 탭 4: 소셜 (Social)

**주요 목적**:
- 바이럴 유도
- 사회적 증거 + 경쟁 심리

**화면 구성**:

1. **친구 초대 섹션**
   - 내 초대 코드
   - “친구가 가입하면 둘 다 XX PHON” 강조
   - Kakao / Instagram / TikTok 공유 버튼

2. **실시간 활동 피드**
   - 친구 또는 전체 유저의 활동 (보상 수령, 스트릭 달성 등)

3. **리더보드**
   - 오늘 / 이번 주 / 전체 랭킹
   - 상위권 유저 노출 (경쟁심 유발)

4. **공유 성과**
   - “이번 주 내가 공유해서 번 PHON”

**심리 설계**:
- 사회적 증거
- 경쟁심
- 바이럴 동기 부여

---

### 탭 5: 프로필 (Profile)

**주요 목적**:
- 소유감 + 진행 상황 확인
- 출금 유도

**화면 구성**:

1. **상단**
   - 프로필 사진 + 닉네임
   - 현재 잔고 (크게 표시)

2. **주요 지표**
   - 총 획득 PHON
   - 최고 스트릭
   - 가입한 지 N일

3. **스트릭 기록**
   - 캘린더 형태로 출석 히스토리

4. **출금 / 설정**
   - “PHON 출금하기” 버튼 (보안 강조)
   - 알림 설정, 계정 관리

**심리 설계**:
- 소유감
- 성취 시각화
- 출금 동기 부여 (돈 버는 실감)

---

## 3. 전체적인 모바일 UX 원칙

- **3초 이내 행동 가능** 구조 유지
- 모든 화면에서 **즉시 피드백** (보상, 애니메이션)
- FOMO 요소를 **홈과 소셜**에 집중 배치
- 한 손 조작 최적화 (하단 CTA, 큰 버튼)
- 감정적 언어 + 이모지 적극 사용 (귀여운 토스트 포함)

---

이 구조는 **한국인 심리 + FOMO + 즉시 보상**을 동시에 자극하면서도,  
모바일에서 가장 자연스럽고 중독성 있게 설계된 탭 구성입니다.
