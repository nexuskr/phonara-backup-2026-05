## 1. 공유 카드 상세 디자인 가이드

### 1.1 기본 철학
공유 카드는 **“자랑하고 싶게 만드는 것”**이 최우선 목표입니다.  
단순한 텍스트가 아니라, **시각적으로 예쁘고 감정적으로 자극적**이어야 합니다.

### 1.2 카드 기본 스펙

- **비율**: 9:16 (Instagram Stories / TikTok 최적화)
- **해상도**: 최소 1080 × 1920px
- **스타일**: Premium Dark + Neon Glow
- **주요 색상**:
  - 배경: #0a0a0f ~ #121218 그라데이션
  - 네온: Cyan (#00f0ff), Purple (#a855f7), Pink (#ff2d95), Gold (#ffd700)

### 1.3 카드 구성 요소 (필수)

| 영역 | 내용 | 비고 |
|------|------|------|
| **상단 로고** | PHONARA 로고 (작게) | 브랜드 인식 |
| **메인 메시지** | 큰 폰트 + 네온 효과 | “오늘 PHON으로 ○○원 벌었어요” |
| **강조 숫자** | 금액 or 일수 (가장 크게) | 시각적 임팩트 |
| **서브 메시지** | 스트릭, 등급, 보너스 등 | 자랑 포인트 |
| **하단 CTA** | “나도 시작하기” + PHONARA 로고 | 바이럴 유도 |
| **네온 효과** | 텍스트 또는 테두리에 glow 적용 | 프리미엄 느낌 |

### 1.4 카드 유형별 예시

**유형 A: 수익 인증형**
- 메인: “오늘 PHON으로 **87,500원** 벌었어요”
- 서브: “스테이킹 수익 + 미션 보상”

**유형 B: 스트릭 인증형**
- 메인: “**14일 연속** 출석 성공!”
- 서브: “매일 조금씩 쌓이는 재미 ”

**유형 C: 입금 보너스형**
- 메인: “첫 입금하고 **15% 보너스** 받았어요 ✨”
- 서브: “소액부터 시작하기 딱 좋아요”

**유형 D: 게임 성과형**
- 메인: “Keepy-Uppy에서 **1위** 했습니다 ”
- 서브: “오늘 게임 보상으로 4,200 PHON”

### 1.5 디자인 가이드라인

- 글씨는 **굵고 명확**하게 (가독성 최우선)
- 네온 효과는 **텍스트와 주요 숫자**에만 적용
- 배경은 너무 화려하지 않게 (정보 전달이 우선)
- 하단 CTA는 항상 포함 (바이럴 유도)

## 2. 게임화 요소 상세 스펙

### 2.1 추천 시스템 게임화

#### 추천 레벨 시스템

| 레벨 | 이름 | 조건 | 혜택 |
|------|------|------|------|
| 1 | 브론즈 추천자 | 추천 성공 1회 | 기본 보상 |
| 2 | 실버 추천자 | 추천 성공 5회 | 추천 보상 +5% |
| 3 | 골드 추천자 | 추천 성공 15회 | 추천 보상 +10% + 특별 뱃지 |
| 4 | 플래티넘 추천자 | 추천 성공 30회 | 추천 보상 +15% + VIP 미션 우선권 |
| 5 | 다이아 추천자 | 추천 성공 50회 | 추천 보상 +20% + 전용 뱃지 + 리더보드 상시 노출 |

#### 추천 미션 예시

- “이번 주에 친구 3명 초대 + 입금 성공 시 **15,000 PHON** 추가 보상”
- “추천으로 10명 유치 시 ‘추천 마스터’ 뱃지 + 영구 보너스”
- “연속 5일 추천 성공 시 스트릭 보너스”

#### 리더보드

- **이번 주 추천왕**
- **이번 달 추천왕**
- 상위 10위까지 공개 + 보상 지급

### 2.2 전체 게임화 요소 종합

| 요소 | 내용 | 목적 |
|------|------|------|
| **추천 레벨** | 브론즈 → 다이아 | 장기 참여 유도 |
| **추천 미션** | 주간/월간 미션 | 목표 의식 부여 |
| **뱃지 시스템** | “첫 추천”, “10명 유치”, “고액 유치” 등 | 성취감 |
| **리더보드** | 주간/월간 추천 랭킹 | 경쟁심 |
| **스트릭** | 연속 추천 성공 일수 | 습관 형성 |
| **진행도 표시** | “친구 4/10명 초대 완료” | 다음 행동 유도 |

## 3. 입금 후 공유 유도 화면 스토리보드

### 화면 1: 입금 완료 화면

**화면 구성**:
- 중앙에 큰 체크 + 네온 애니메이션
- 텍스트: **“입금이 완료되었습니다!”**
- 하단에: “**15% 보너스 PHON**이 지급되었어요 ✨”

**버튼**:
- [메인] **공유 카드 보기**
- [서브] **홈으로 가기**

---

### 화면 2: 공유 카드 미리보기 + 공유 유도

**화면 구성**:
- 상단: “이 기쁨을 친구에게도 알려주세요!”
- 중앙: **공유 카드 미리보기** (예쁘게 생성된 카드)
- 하단 버튼 3개:
  1. **카카오톡으로 공유하기** (가장 크게)
  2. **인스타그램 스토리 공유**
  3. **링크 복사**

**추가 요소**:
- 작게: “공유하면 친구도 보너스를 받고, 너도 추가 보상을 받아요”

---

### 화면 3: 공유 성공 화면 (선택)

**조건**: 사용자가 실제로 공유를 완료했을 때 노출

**내용**:
- “공유 감사합니다!”
- “추가로 **1,000 PHON**을 드렸어요”
- 버튼: [홈으로 가기] / [추천 코드 다시 보기]

---

### 화면 4: 공유 안 함 선택 시

**조건**: 사용자가 “나중에 하기”를 선택했을 때

**내용**:
- “언제든 프로필에서 공유할 수 있어요”
- “친구가 입금하면 너도 보상을 받아요”
- 버튼: [추천 코드 보기] / [홈으로]

---

### 전체 흐름 요약
입금 완료
↓
공유 카드 미리보기 + 공유 유도
↓
공유 성공 → 추가 보상 지급
OR
나중에 하기 → 나중에 공유 유도
